# Working Notes

This directory contains in-progress working notes for active tasks. Agents should update these as they work so the next agent (or human) can pick up where they left off.

## Convention

- File name: `<stream-or-task>-<topic>.md` (e.g., `stream-3-auth-spike.md`)
- Each note starts with a **Status** line and **Last Updated** date
- When a task completes, the note can be archived or deleted — the decision should already be captured in `docs/decisions/`
