# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the NanoClaw → Copilot SDK migration.

ADRs record decisions that are **done** — the context, the decision, and the consequences. They don't change once written (superseded decisions get a status update, not an edit).

## Format

Each ADR follows this template:
- **Status**: Accepted / Superseded / Deprecated
- **Context**: What prompted the decision
- **Decision**: What we decided
- **Consequences**: What follows from the decision
